# django-sugar

### 安装

```sh
# 安装稳定版 (推荐)
pip3 install django-sugar==0.1.0

# 安装开发版
pip3 install git+https://github.com/yingzhuo/django-sugar.git@main
```

### 必要依赖

* `pip3 install django`
* `pip3 install djangorestframework`
* `pip3 install pyjwt`
* `pip3 install cryptography`

### 可选依赖

* `pip3 install bcrypt`
