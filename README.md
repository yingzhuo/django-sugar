# django-sugar

### 安装

```sh
# install
pip3 install git+https://github.com/yingzhuo/django-sugar.git@main

# install (upgrade)
pip3 uninstall -y django-sugar && \
  pip3 install git+https://github.com/yingzhuo/django-sugar.git@main
```

### 必要依赖

* `pip3 install django`
* `pip3 install djangorestframework`
* `pip3 install pyjwt`
* `pip3 install cryptography`

### 可选依赖

* `pip3 install bcrypt`
